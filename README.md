# The Enchanted Interpreter

A cozy, story-driven Python course for beginners.

---

## Welcome to Willowmere
You've inherited your grandmother's cottage — and with it, a responsibility you never expected.
Deep beneath the cottage lies a cellar, its walls lined with ancient sigils that once protected the village of Willowmere. With your grandmother gone, the magic has begun to fade and the wards are failing.
To restore them, you'll need to master the art of Pythonic magic.

---

## What Is This
The Enchanted Interpreter is a beginner Python course that teaches programming through a cozy fantasy story. Each 'sigil' introduces a new concept, a piece of the story, and a challenge to restore a faded sigil. Complete all the sigils in a room and perform the ritual — a larger challenge that ties everything together.
The video lessons live on YouTube. This repository contains everything else:

- Story content for each chapter
- Challenge briefs (sigils and rituals)
- Starter code where relevant

You don't need to follow the story to use this course — but it's more fun if you do.

## How To Use This Repo
Each room has its own folder, broken into sigils. Inside each sigil you'll find:

- narrative.md — the narrative content for that chapter
- lesson.py - the lesson file used in the video
- sigil.md — your challenge
- starter.py — starter code where provided
- solution.py - my solution to the challenge *Note: There are often a million ways to achieve the same end in programming. Don't be discouraged if you took a different approach than I did. My solution is just one of many*


Each room folder also contains a `ritual/` folder with the rooms 'ritual' or challenge.

### The recommended path:

1. Watch the YouTube lesson for the chapter
2. Read the story
3. Complete the sigil challenge
4. Once all sigils in a module are done, perform the ritual
